# pieces-of-code
Some pieces of code that can be of use on their own or as part of larger applications.<br>
You can check out the input and output that can be found in separate files or in comments at the end of the code/script. <br>
Copyright notices are included separately for the respective files.<br>
I know that including copyright notices may be excessive for basic code, but according to copyright law, rights are reserved by default if nothing is mentioned and I didn't want anyone to think I was claiming rights for basic code.<br>
I try to follow good and consistent practices most of the time. The coding and file naming conventions I followed align with what I understand to be good practices, but I am open to following other conventions if needed.
