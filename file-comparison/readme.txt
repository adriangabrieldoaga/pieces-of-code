Commands for byte by byte file comparison. Useful to detect corrupted files.
Works with any type of file (image, video, text etc.).
Alternative to using programs such as Total Commander or Directory Opus.
The one including all subfolders, windows-command-prompt-compare-the-files-in-two-folders-including-all-subfolders-byte-by-byte-and-print-output-to-a-text-file.bat, seems
to have some limits, depending on the files.
cpp-file-comparison-summary.cpp finds the lines in the output text file that tell you about files that have differences.
